# 04 20-Sample Test Vectors

## Purpose

This step stores a compact end-to-end verification payload for the SystemVerilog testbench.

The 20-sample scope is intentionally small enough to inspect manually while still exercising the full forward path from `embedding_output` to `logits` and `pred_class`.

## Files

```text
seed1234/fixed/embedding_output_20.txt
seed2026/fixed/embedding_output_20.txt
```

Input embeddings for the PL golden model. Each row represents one flattened sample.

```text
seed1234/fixed/logits_20.txt
seed2026/fixed/logits_20.txt
```

Expected fixed-point logits generated by the PyTorch reference.

```text
seed1234/fixed/pred_class_20.txt
seed2026/fixed/pred_class_20.txt
```

Expected categorical predicted classes.

```text
seed1234/fixed/label_20.txt
seed2026/fixed/label_20.txt
```

Dataset labels for reference/debugging. These are not the same thing as RTL correctness; the RTL is checked against the Python reference.

```text
seed1234/fixed/sample_count.txt
seed2026/fixed/sample_count.txt
```

Number of samples in the payload.

## How To Regenerate

From `../02_fixed_point_export/`:

```text
python dump_pl_io_all_samples.py
```

## Result Of This Step

```text
20 fixed-point input/output samples -> compact payload for RTL verification
```
